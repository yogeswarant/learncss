display - block, inline, inline-block, none

display: block

Do inspect element and check that the div is occupying the entire horizontal space. This the default property when display is block.

display: block;

display: inline-block

Now the div will not occupy the entire horizontal space. This the default property when display is inline-block.

Note: Even div can be given inline, but the elements inside the div will not be displayed as expected. Thats the reason to use inline-block.

display: inline-block;

display: inline

This is a span test - which has display: inline by default This is an anchor tag - which has display: inline by default

display: none

The contents inside div whose display is set as none will not be displayed.
This can be used for hiding without taking space. If you want to hide and but still use the space then use visibility: hidden.

display: none;
