Box model - Padding

Padding


padding: [top&bottom] [left&right]
padding: 20px 40px;

or

padding-left: 40px;
padding-right: 40px;
padding-top: 20px;
padding-bottom: 20px;



padding will go beyond the width. The total width will be width + padding.

To be more specific, total width = width + border + padding
