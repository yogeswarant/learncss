Relative font size

Contents inside main
Contents inside smaller - smaller than main


Contents inside main
Contents inside sub - 90% of main

The size can be reduced either by using smaller or given % of parent size.

  	.main {
		padding: 20px;
		margin: 0 auto;
		background-color: #587ff5;
		max-width: 200px;
		border-radius: 5px;
	}

	.smaller {
		font-size: smaller;
	}

	.sub {
		font-size: 90%;
	}
